# 瞳虎-银行卡OCR MCP 服务文档

## 什么是 瞳虎-银行卡OCR MCP 服务？
瞳虎银行卡OCRMCP Server提供以下核心功能：
- 银行卡OCR。

服务特点：
- **覆盖信息全**：精准识别银行卡图片上的文字信息，包括银行卡号、卡类型、银行名称等信息。
- **数据精准**：识别率高达99.9%以上。

## 如何使用 瞳虎-银行卡OCR MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【银行卡OCR】服务
### 部署方式1（SSE）
```
{
  "mcpServers": {

      "url": "https://mcp.tonghu.top/sse?apiKey=您在瞳虎mcp平台上申请的apiKey"

```

## 瞳虎-银行卡OCR MCP 服务的使用案例

1. **案例 1：第三方支付平台新用户绑卡场景**  
   - 背景：某第三方支付平台新用户绑卡注册时，手动输卡号错误率达 12%，导致 30% 用户流失。
   - 具体流程：用户上传银行卡照片，AI平台调用 MCP 接口识别卡号、开户行等信息，让用户确认，确认后完成绑卡。
   - 成效：卡号输入错误率降至 0.1%，用户绑卡时长从 120 秒缩短至 30 秒，新用户注册转化率提升 8%，每月减少近万次用户流失。

3. **案例 2：银行信用卡中心线下办卡场景**  
   - 背景：某银行信用卡中心线下办卡，工作人员手动录入用户银行卡信息，平均每张卡耗时 5 分钟，且错误率达 8%。
   - 具体流程：工作人员用平板拍摄用户银行卡，AI调用 MCP 接口实时识别信息，并自动填充至办卡系统，核对无误后提交，完成信息采集。
   - 成效：单张卡信息录入时长缩短至 1 分钟，错误率降至 0.5%，每日线下办卡量从 100 张提升至 350 张，客户等待满意度提升 12%。
  
4. **案例 3：互联网保险公司投保扣费场景**  
   - 背景：某互联网保险公司投保流程中，用户手动绑卡扣保费，失败率达 10%，因卡号错误引发的投诉占比 60%，投保完成率低。
   - 具体流程：用户投保至绑卡环节，上传银行卡照片，MCP 接口识别并校验卡号与开户行匹配性，校验通过后自动绑定，发起保费扣款。
   - 成效：绑卡失败率降至 1.5%，相关投诉量减少 60%，投保流程完成率从 75% 提升至 85%，每月新增有效投保用户超 3000 人。
