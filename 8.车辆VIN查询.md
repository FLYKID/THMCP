# 瞳虎-车辆VIN查询 MCP 服务文档

## 什么是 瞳虎-车辆VIN查询 MCP 服务？
瞳虎车辆VIN查询MCP Server提供以下核心功能：
- 传入车辆识别码（车架号）VIN码来获取车辆的品牌名称、车型名称、出厂日期、发动机类型、排量、车系等多维度车辆参数。

服务特点：
- **覆盖信息全**：可获取车辆的品牌名称、车型名称、出厂日期、发动机类型、排量、车系等多维度车辆参数。
- **实时更新**：非缓存数据，直连官方数据，支持大小车及新能源车辆。

## 如何使用 瞳虎-车辆VIN查询 MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【车辆VIN查询】服务
### 部署方式1（SSE）
```
{
  "mcpServers": {
    "TH_MCP": {
      "url": "https://mcp.tonghu.top/sse?apiKey=您在瞳虎mcp平台上申请的apiKey"
    }
  }
}
```
### 部署方式2（Streamable Http）
