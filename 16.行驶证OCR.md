# 瞳虎-行驶证OCR MCP 服务文档

## 什么是 瞳虎-行驶证OCR MCP 服务？
瞳虎行驶证OCR MCP Server提供以下核心功能：
- 行驶证OCR。

服务特点：
- **覆盖信息全**：传入行驶证图片或url地址，智能识别行驶证图片上的文字信息，返回包括所有人、签发机关、品牌型号、住址、车牌号、品牌型号、发动机号码、车架号、车辆类型、注册日期、发证日期、使用性质等信息，兼容多种图片格式。
- **数据精准**：性能稳定，识别率高，准确率高，速度快。

## 如何使用 瞳虎-行驶证OCR MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【行驶证OCR】服务
### 部署方式1（SSE）
```
{
  "mcpServers": {
    "TH_MCP": {
      "url": "https://mcp.tonghu.top/sse?apiKey=您在瞳虎mcp平台上申请的apiKey"

```

## 瞳虎-行驶证OCR MCP 服务的使用案例

**案例1：二手车交易行业 - 车辆信息录入与核验场景**
	- 背景：二手车门店收车/交易时需人工录入行驶证信息，耗时久易出错，核验车辆合规性效率低，影响交易流转。

	- 具体流程：店员用搭载接口的智能终端拍摄行驶证，实时识别品牌、车架号等信息自动填入系统，同步比对车管所备案信息完成初步核验。

	- 成效：单台车信息录入时间从2分钟缩至20秒，录入错误率降至0.3%以下，合规核验效率提升65%，单店日交易量提升20%。

**案例2：货运物流行业 - 车辆调度前合规核验场景**

	- 背景：物流企业调度车辆前需核实行驶证年检、营运资质等信息，人工核对繁琐，易遗漏过期证件，存在运输合规风险。

	- 具体流程：调度员通过企业管理系统上传行驶证照片，调用MCP接口识别关键信息，系统自动校验年检有效期、营运范围，生成合规性报告。

	- 成效：单台车合规核验时间从1.5分钟缩至10秒，过期证件/不合规车辆识别准确率100%，运输合规风险降低85%，调度效率提升70%。

**案例3：车险理赔行业 - 报案车辆信息核对场景**

	- 背景：车险报案后需人工核对行驶证车辆信息与保单一致性，耗时久易出错，导致理赔受理延迟，用户投诉率高。

	- 具体流程：用户APP上传行驶证照片，后端调用MCP接口识别信息，自动与保单系统匹配比对，异常信息提示人工复核，正常信息直接进入理赔流程。

	- 成效：理赔信息核对自动化率达90%，单案核对时间从3分钟缩至15秒，理赔受理周期压缩至30分钟内，用户投诉率下降68%。
