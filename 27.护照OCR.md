# 瞳虎-护照OCR MCP 服务文档

## 什么是 瞳虎-护照OCR MCP 服务？
瞳虎护照OCR MCP Server提供以下核心功能：
**中国大陆护照OCR**
	- 支持对中国大陆居民护照的资料页进行结构化识别，包含国家码、姓名、性别、护照号、出生日期、签发日期、有效期至、签发地点。
**港澳台证件OCR**
	- 支持识别4类港澳台出入境证件，包含港澳通行证正/反面、台湾通行证正/反面、台胞证（台湾居民来往大陆通行证）正/反面、返乡证（港澳居民来往内地通行证）正/反面。
**港澳台地区及境外护照OCR**
	- 支持对港澳台地区及境外护照进行结构化识别，包括MRZCode1、MRZCode2、出生日期、国家码、国籍、姓名拼音、性别、护照号、护照类型、有效期。

服务特点：
- **覆盖信息全**：支持对中国大陆护照、港澳台地区及境外护照、港澳台证件（港澳通行证、台湾通行证、台胞证、返乡证）进行结构化识别。
- **数据精准**：性能稳定，识别率高，准确率高，速度快。

## 瞳虎-护照OCR MCP 服务的使用案例

**案例1：出入境管理局 - 护照办理/续签信息录入场景**

	- 背景：群众办理护照申领或续签时，工作人员需人工录入护照号、姓名、国籍、有效期等信息，项多易出错，耗时久导致排队等待时间长。

	- 具体流程：工作人员用政务终端扫描群众护照原件，调用MCP接口实时识别护照号、姓名、有效期、签发机关等关键信息，自动录入出入境管理系统，同步校验信息完整性。

	- 成效：单笔护照业务信息录入时间从3分钟缩至30秒，录入错误率降至0.1%以下，窗口办理效率提升70%，群众平均等待时间减少65%，日均办理量提升55%。

**案例2：国际酒店 - 境外旅客入住登记场景**

	- 背景：境外旅客入住酒店时，前台需人工登记护照信息并上传公安系统，信息繁琐易出错，登记耗时久导致旅客等待体验差，且存在信息报送延迟风险。

	- 具体流程：前台工作人员用登记终端扫描旅客护照，调用MCP接口识别姓名、护照号、入境日期等信息，自动填入酒店PMS系统，同步推送至公安境外人员登记平台。

	- 成效：单名境外旅客入住登记时间从2分钟缩至15秒，信息登记错误率降至0.05%以下，公安系统信息报送及时率达100%，旅客满意度提升72%，前台接待效率提升80%。

**案例3：跨境企业 - 外籍员工备案审核场景**

	- 背景：跨境企业为外籍员工办理工作许可、社保备案时，需人工核对护照信息，海量材料处理耗时久、易出错，导致备案进度延迟，影响员工合法履职。

	- 具体流程：HR收集外籍员工护照照片，批量上传至企业合规管理系统，调用MCP接口批量识别护照关键信息，自动与工作许可申请信息比对审核，生成备案材料清单。

	- 成效：单批次30名外籍员工备案信息审核时间从1.5小时缩至10分钟，审核错误率为0，备案流程效率提升90%，备案进度延迟问题彻底解决，员工合法履职保障更及时。
