## 如何使用 瞳虎-发票OCR MCP 服务？

1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）

## 瞳虎-发票OCR MCP 服务的使用案例

1. **电商企业月度发票批量处理**  
   - 背景：某电商企业每月订单超 2 万单，需处理大量买家发票，人工录入易出错，单月耗时超 100 工时。
   - 具体流程：客户接入发票OCR MCP服务，AI订单系统自动导出待开发票信息，上传发票图片至接口，识别后数据同步至财务系统，自动生成凭证。
   - 发票处理效率提升 80%，单月工时缩减至 20 小时内，数据错误率从 5% 降至 0.1%，减少财务返工成本。

3. **物流企业司机报销发票管理**  
   - 背景：某物流公司有 500 + 司机，每月报销加油、路桥发票超 3000 张，人工核验慢且易漏假票。
   - 具体流程：司机通过企业 APP 上传发票照片，AI处理系统调用 MCP 接口实时识别发票信息并验真，数据同步至财务报销系统，财务仅复核异常单据。
   - 报销审核周期从 7 天缩短至 2 天，假票识别率达 99%，财务核验工作量减少 80%。
  
4. **连锁餐饮门店采购发票对账**  
   - 背景：某连锁餐饮品牌有 80 家门店，每月采购发票超 5000 张，总部需人工核对发票与采购单，易出现数据偏差且耗时久。
   - 具体流程：门店上传采购发票至总部系统，AI调用 MCP 接口提取发票关键信息，与系统内采购单自动比对，生成差异报表供财务确认。
   - 成效：对账效率提升 75%，每月对账时间从 10 天压缩至 3 天，数据偏差率从 6% 降至 0.8%，降低财务对账成本。
