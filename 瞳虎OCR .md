# 瞳虎 OCR MCP 服务文档

## 什么是 瞳虎 OCR MCP 服务？
瞳虎OCR MCP Server提供以下核心功能：
- **发票财务票据OCR**
  - 支持票据混贴，支持常见的十余种票据智能智能识别，包括增值税发票、卷票、机打发票、定额发票、火车票、出租车票、网约车行程单、飞机行程单、汽车票、过路过桥费、船票、机动车/二手车销售发票 等等。
- **营业执照OCR**
  - 上传营业执照，解析营业执照上的法人、注册资本、有效期、地址、社会信用代码、公司名称、注册日期等信息。
- **银行卡OCR**
  - 精准识别银行卡图片上的文字信息，包括银行卡号、卡类型、银行名称等信息。
- **身份证OCR**
  - 传入身份证照片，识别照片文字信息并返回，包括姓名、身份证号码、性别、民族、出生年月日、地址、签发机关及有效期。支持正反面识别。
- **驾驶证OCR**
  - 传入驾驶证图片，识别驾驶证基本信息，识别驾驶证主页的上的姓名、出生年月、性别、地址、国籍、准驾车型、驾驶证号、驾驶证正本版本、初次领证日期、签发机关、有效日期、有效年限、驾驶证副业上等文字信息，识别准确率高达99.9%以上，性能更稳定、识别更精准。
- **行驶证OCR**
  - 传入行驶证图片或url地址，智能识别行驶证图片上的文字信息，返回包括所有人、签发机关、品牌型号、住址、车牌号、品牌型号、发动机号码、车架号、车辆类型、注册日期、发证日期、使用性质等信息，兼容多种图片格式，性能稳定，识别率高，准确率高，速度快，支持多种图片形式。
- **车牌号OCR**
  - 传入车牌号照片或url地址，扫描识别车牌号和车牌颜色，支持识别机动车蓝牌、黄牌、绿牌、新能源、港澳出入境车牌、农用车牌等多种车牌号，支持同时识别多个车牌，支持自动旋转，性能稳定，识别率高。
- **车架号OCR**
  - 车架号OCR、VIN码识别。支持对车辆挡风玻璃、发动机铭牌处的车架号码(VIN码)进行精准识别。
- **车辆合格证OCR**
  - 智能识别上传车辆合格证图片信息，包括合格证编号、发证日期、车辆制造企业名、车辆品牌、车辆名称、车辆型号、车架号、车身颜色、发动机型号、发动机号、燃料种类、排量、功率、排放标准、轮胎数、轴距、轴数、转向形式、总质量、整备质量、驾驶室准乘人数、最高设计车速、车辆制造日期等信息。
- **机动车登记证OCR**
  - 智能识别上传机动车登记证书图片信息，包括编号、机动车所有人、登记机关、登记日期、登记编号、车辆类型等信息。支持各类图片格式。
- **快递面单OCR**
  - 可识别市面上常见的快递面单，包括四通一达/顺丰/京东/邮政/极兔/天天等快递面单，返回运单号、收/寄件人姓名、收/寄件人电话、收/寄件人地址等字段信息
- **出生证明OCR**
  - 传入出生医学证明图片，可识别出生医学证明的关键字段，包括新生儿姓名、性别、出生时间、父亲姓名、母亲姓名等。广泛应用于新生儿身份登记、入学登记、生育险报销等场景。
- **户口本OCR**
  - 识别户口本图像文字信息，并返回出生地址、名称、与户主关系、身份证号、性别、民族、婚姻状态、登记日期、出生日期、学历、籍贯等信息。精准识别，支持多种图片格式。
- **房产证OCR**
  - 传入房产证图片，支持对房产证进行结构化识别，包括权利人、坐落、权利类型、面积、字第号 、不动产单元号、共有情况、用途、使用期限 、登记日期、共有人等字段。
- **社保卡OCR**
  - 传入社保卡图片，精准识别社保卡上的文字信息，包括社保卡卡号、社会保障号码、签发日期、持卡人姓名、有效期限等。
- **结婚证OCR**
  - 支持对结婚证进行结构化识别，包括姓名、身份证件号、出生日期、国籍、性别、结婚证字号、持证人、备注、登记日期。
- **护照OCR**
  - 中国大陆护照识别、港澳台地区及境外护照识别、港澳台证件识别（港澳台出入境证件，包含港澳通行证、台湾通行证、台胞证、返乡证）
  
服务特点：
- **覆盖信息全**：支持各种常见发票、营业执照、银行卡、身份证等等各种场景的结构化识别。
- **数据精准**：识别率高达99.9%以上。
