# 瞳虎-营业执照OCR MCP 服务文档

## 什么是 瞳虎-营业执照OCR MCP 服务？
瞳虎营业执照OCRMCP Server提供以下核心功能：
- 营业执照OCR。

服务特点：
- **覆盖信息全**：上传营业执照，解析营业执照上的法人、注册资本、有效期、地址、社会信用代码、公司名称、注册日期等信息
- **数据精准**：识别率高达99.9%以上。

## 如何使用 瞳虎-营业执照OCR MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【营业执照OCR】服务
### 部署方式1（SSE）
```
{

      "url": "https://mcp.tonghu.top/sse?apiKey=您在瞳虎mcp平台上申请的apiKey"

  }
}
```

## 瞳虎-营业执照OCR MCP 服务的使用案例

1. **案例 1：企业服务公司代注册业务**  
   - 背景：某企业服务公司承接代注册业务，人工录入营业执照信息，日均处理 30 单，效率低且易出错。
   - 具体流程：对接营业执照 OCR MCP，员工上传客户执照图片，系统自动识别名称、信用代码等信息，生成表单后自动核对。
   - 成效：日均处理量提升至 100 单，信息错误率从 15% 降至 0.1%，单客户服务时长从 40 分钟缩短至 10 分钟。

3. **案例 2：城商行小微企业信贷审核**  
   - 背景：某城商行开展小微企业信贷，人工核验营业执照需逐页核对信息、查工商库，单户审核耗时超 3 小时，客户等待久。
   - 具体流程：客户线上上传执照，银行系统调用 OCR 接口自动提取关键信息，同步与工商数据库比对，生成核验报告供审核员参考。
   - 成效：单户执照审核时长缩至 25 分钟，审核效率提升 86%，客户因等待过长取消申请的比例从 22% 降至 5%。
  
4. **案例 3：家居垂直电商平台招商**  
   - 背景：某家居电商平台招商，大量商家提交营业执照，人工录入企业名称、地址等信息，日均处理 120 家，易漏填、错填，影响入驻进度。
   - 具体流程：商家在入驻页面上传执照，平台调用 MCP 接口自动识别并填充信息至入驻表单，审核员仅复核重点字段。
   - 成效：日均处理商家数提升至 300 家，信息漏填率从 18% 降至 2%，商家从提交材料到完成入驻的周期从 3 天缩至 1 天。
