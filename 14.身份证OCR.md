# 瞳虎-身份证OCR MCP 服务文档

## 什么是 瞳虎-身份证OCR MCP 服务？
瞳虎身份证OCRMCP Server提供以下核心功能：
- 身份证OCR。

服务特点：
- **覆盖信息全**：传入身份证照片，识别照片文字信息并返回，包括姓名、身份证号码、性别、民族、出生年月日、地址、签发机关及有效期。支持正反面识别。
- **数据精准**：识别率高达99.9%以上。

## 瞳虎-身份证OCR MCP 服务的使用案例

1. **案例 1：快递网点实名制登记**  
   - 背景：快递高峰期网点人工登记身份证，每单耗时 5 分钟，错填信息导致错单率达 3%，客户投诉频发。
   - 具体流程：快递员用手持终端扫描客户身份证，终端调用 OCR 接口实时识别信息，自动同步至快递系统，客户确认信息无误即可完成登记。
   - 成效：单票登记时间缩至 30 秒，错单率降至 0.1%，每月减少投诉 15 起，网点日均处理量提升 50%，节省 2 名人工成本。

3. **案例 2：职业培训学校新生报名**  
   - 背景：新生报名时人工核验身份证、录入信息，每人需 10 分钟，信息误差率 4%，导致后续学籍注册反复修改，影响开课进度。
   - 具体流程：报名处工作人员用扫码设备扫描新生身份证，系统调用 MCP 接口提取信息并自动填入报名系统，与学籍库初步比对后确认提交。
   - 成效：单人生成报名信息时间缩至 1 分钟，信息误差率降至 0.2%，学籍注册修改次数减少 90%，单日报名接待量从 80 人增至 200 人。
  
4. **案例 3：汽车租赁公司身份核验**

   - 具体流程：客户在租车 APP 上传身份证照片，系统调用 MCP 接口完成信息识别与真伪校验，同步比对征信数据，通过后即可办理租车。
   - 成效：单客核验时间缩至 3 分钟，假证识别率达 99.9%，近 3 个月无租骗纠纷，人工核验成本减少 40%，租车订单转化率提升 25%。
