# 瞳虎-出生证明OCR MCP 服务文档

## 什么是 瞳虎-出生证明OCR MCP 服务？
瞳虎出生证明OCR MCP Server提供以下核心功能：
- 出生证明OCR。

服务特点：
- **覆盖信息全**：传入出生医学证明图片，可识别出生医学证明的关键字段，包括新生儿姓名、性别、出生时间、父亲姓名、母亲姓名等。广泛应用于新生儿身份登记、入学登记、生育险报销等场景。
- **数据精准**：性能稳定，识别率高，准确率高，速度快。

## 瞳虎-出生证明OCR MCP 服务的使用案例

**案例1：妇幼保健院 - 新生儿信息建档与证件办理场景**
	
	- 背景：妇幼保健院办理出生证明后需人工录入新生儿姓名、出生日期、父母信息等建档，信息项多易出错，耗时久导致家长排队等待时间长。

	- 具体流程：医护人员用智能终端拍摄出生证明，调用MCP接口实时识别新生儿及父母关键信息，自动录入医院信息系统完成建档，同步生成办理后续证件的信息清单。

	- 成效：单份出生证明信息建档时间从3分钟缩至30秒，录入错误率降至0.1%以下，家长平均等待时间减少60%，医院日均建档办理量提升50%。

**案例2：户籍办理机构 - 新生儿户口登记审核场景**

	- 背景：家长为新生儿办理户口时，工作人员需人工核对出生证明信息与申报材料一致性，流程繁琐易出错，导致户口登记效率低，家长多次往返。

	- 具体流程：工作人员扫描家长提交的出生证明，调用MCP接口识别信息，系统自动与家长提交的户口申报信息比对核验，生成审核结果，信息一致则进入登记流程。

	- 成效：单份户口登记信息核对时间从2分钟缩至15秒，审核准确率达99.8%，家长因信息核对问题往返率降至0.5%以下，户口登记效率提升75%。

**案例3：少儿医保经办机构 - 参保资质审核场景**

	- 背景：家长为新生儿办理少儿医保时，需提交出生证明审核参保资质，人工核对信息耗时久、易遗漏关键信息，导致参保审核周期长。

	- 具体流程：家长通过医保线上参保平台上传出生证明照片，后台调用MCP接口识别新生儿信息、父母参保信息等，自动比对参保条件，完成初步资质审核。

	- 成效：单份少儿医保参保初审时间从5分钟缩至40秒，初审准确率达99.7%，参保审核整体周期从7个工作日缩短至2个工作日，家长线上办理满意度提升80%。
