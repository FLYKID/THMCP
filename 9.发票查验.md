# 瞳虎-发票查验 MCP 服务文档

## 什么是 瞳虎-发票查验 MCP 服务？
瞳虎发票查验MCP Server提供以下核心功能：
- 输入发票基本信息：发票代码、发票号码、开票日期、校验码后6位、不含税金额、含税金额，核验发票真伪。

服务特点：
- **覆盖信息全**： 支持机动车、二手车销售发票、航空运输电子客票、铁路电子客票等各种类型发票。
- **实时更新**：非缓存数据，直连官方数据，毫秒级响应。

## 如何使用 瞳虎-发票查验 MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【发票查验】服务
