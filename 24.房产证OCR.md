# 瞳虎-房产证OCR MCP 服务文档

## 什么是 瞳虎-房产证OCR MCP 服务？
瞳虎房产证OCR MCP Server提供以下核心功能：
- 房产证OCR。

服务特点：
- **覆盖信息全**：传入房产证图片，支持对房产证进行结构化识别，包括权利人、坐落、权利类型、面积、字第号 、不动产单元号、共有情况、用途、使用期限 、登记日期、共有人等字段。
- **数据精准**：性能稳定，识别率高，准确率高，速度快。
