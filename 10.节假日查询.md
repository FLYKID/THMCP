# 瞳虎-节假日查询 MCP 服务文档

## 什么是 瞳虎-节假日查询 MCP 服务？
瞳虎节假日查询MCP Server提供以下核心功能：
- 查询指定年份的所有假期调休日。
- 查询指定年份的所有节假日。
- 查询指定日期是否为放假日期以及节日种类、期限、调班等信息。

服务特点：
- **覆盖信息全**：可查询节假日列表，节假日数据和调休日列表。
- **实时更新**：非缓存数据，毫秒级响应。

## 如何使用 瞳虎-节假日查询 MCP 服务？
### API Key 获取方式
1. 注册登录[瞳虎MCP平台](https://mcp.tonghu.top "瞳虎MCP")
2. 创建您的 API Key（已创建的可直接使用）
3. 在[产品中心](https://mcp.tonghu.top/#/layout/prodCenter "瞳虎MCP")开通【节假日查询】服务

## 瞳虎-节假日查询 MCP 服务的使用案例

1. **企业内部排班管理**  
   - 在制定月度或年度排班表时，集成 MCP 节假日查询服务以自动识别节假日和调休日。
   - 根据查询结果调整员工的工作时间和休息时间，确保符合劳动法规要求。
   - 精确安排员工出勤，避免违规操作。提升员工满意度，促进团队和谐。

3. **电商平台促销活动策划**  
   - 提前通过 MCP 查询未来几个月内的所有节假日，并据此设计促销策略。
   - 针对不同节日特点定制营销方案，如情人节特惠、国庆大促等。
   - 抓住销售高峰，增加销售额。
  
4. **旅游出行预订系统**  
   - 将 MCP 查询功能嵌入到预订系统中，允许用户查看选定日期是否为节假日及其可能带来的影响。
   - 根据节假日情况推荐合适的旅行目的地和行程安排。
   - 提供个性化服务，改善用户体验。帮助旅客避开高峰期，享受更舒适的旅程。
