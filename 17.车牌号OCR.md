# 瞳虎-车牌号OCR MCP 服务文档

## 什么是 瞳虎-车牌号OCR MCP 服务？
瞳虎车牌号OCR MCP Server提供以下核心功能：
- 车牌号OCR。

服务特点：
- **覆盖信息全**：传入车牌号照片或url地址，扫描识别车牌号和车牌颜色，支持识别机动车蓝牌、黄牌、绿牌、新能源、港澳出入境车牌、农用车牌等多种车牌号，支持同时识别多个车牌，支持自动旋转。
- **数据精准**：性能稳定，识别率高，准确率高，速度快。

## 瞳虎-车牌号OCR MCP 服务的使用案例

**案例1：智慧停车行业 - 车辆出入自动计费场景**

	- 背景：传统停车场依赖人工登记车牌收费，高峰期拥堵严重，易出现车牌登记错误、漏收费等问题，运营效率低。

	- 具体流程：停车场出入口摄像头抓拍车辆画面，调用MCP接口实时识别车牌，系统自动匹配入场时间生成计费信息，车辆离场时自动完成费用结算。

	- 成效：车辆出入通行时间从15秒缩至3秒，高峰期拥堵率下降90%，车牌识别准确率达99.8%，漏收费率降至0.1%以下，单停车场人工成本减少60%。

**案例2：城市交通管理 - 违章停车自动取证场景**

	- 背景：交警人工巡查取证违章停车效率低，覆盖范围有限，难以应对城市全域管控需求，违章处理周期长。

	- 具体流程：道路监控设备自动抓拍违停车辆，调用MCP接口快速识别车牌，系统同步关联车辆登记信息，自动生成含车牌、时间、地点的违章取证报告推送至交管平台。

	- 成效：单条违章取证时间从5分钟缩至30秒，城市交通管控覆盖范围提升3倍，违章处理周期从3天缩至1天，违停发生率下降45%，交警巡查人力成本减少50%。

**案例3：网约车平台 - 司机车辆合规核验场景**

	- 背景：网约车平台审核司机车辆资质时，需人工核对车牌与备案信息一致性，易出现“套牌”“错配”问题，存在合规运营风险。

	- 具体流程：司机在平台上传车辆正面照片，调用MCP接口识别车牌，系统自动与平台备案的司机车辆信息库比对，核验通过后方可激活接单权限。

	- 成效：单辆车资质核验时间从2分钟缩至10秒，车牌信息比对准确率达100%，成功拦截98%的“套牌”“错配”违规车辆，平台合规运营风险降低90%，审核效率提升85%。
